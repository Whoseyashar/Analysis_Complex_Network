import networkx as nx
import matplotlib.pyplot as plt


edges_file_path = 'C:/Users/Vampire/Desktop/complex/facebook/1684(1).edges'  

# I Initialize an undirected graph for Facebook-Ego network
G = nx.Graph()

# Then I Read edges from the file and add them to the graph
with open(edges_file_path, 'r') as f:
    for line in f:
        node1, node2 = map(int, line.strip().split())
        G.add_edge(node1, node2)

# Plotting degree distribution
degrees = [G.degree(n) for n in G.nodes()]
plt.figure(figsize=(10, 6))
plt.hist(degrees, bins=range(1, max(degrees)+1), density=True, alpha=0.75, color='blue')
plt.title("Degree Distribution of the Facebook-Ego Network")
plt.xlabel("Degree")
plt.ylabel("Frequency")
plt.grid(True)
plt.show()

# Get the number of nodes and edges in the original network
num_nodes = G.number_of_nodes()

# Generate a random graph with the same number of nodes and edges
random_graph = nx.gnm_random_graph(num_nodes, num_edges)

# Degree distribution of the random graph
random_degrees = [random_graph.degree(n) for n in random_graph.nodes()]

# Plotting the degree distribution of the random network
plt.figure(figsize=(10, 6))
plt.hist(random_degrees, bins=range(1, max(random_degrees)+1), density=True, alpha=0.75, color='green', label='Random Network')
plt.hist(degrees, bins=range(1, max(degrees)+1), density=True, alpha=0.5, color='blue', label='Facebook-Ego Network')
plt.title("Comparison of Degree Distribution: Facebook-Ego vs Random Network")
plt.xlabel("Degree")
plt.ylabel("Frequency")
plt.legend()
plt.grid(True)
plt.show()

Main difference between the Facebook network and a random network: As we can observe the plot,in Facebook-Ego network The distribution is more skewed, with a few nodes having significantly higher degrees. On the other hand,in Random network The degree distribution is more concentrated around a central value, which is expected for random graphs. As a whole,The Facebook network has a power-law distribution, meaning a few nodes have many connections, while most have only a few. This is typical of scale-free networks, where connections grow through preferential attachment some nodes attract more connections over time. In contrast, a random network shows a more uniform or normal distribution, where most nodes have a similar number of connections, since edges are formed randomly without preference. In my idea the reason for this difference is that, in real-world networks like Facebook, some people (consider as nodes) naturally attract more connections over time, a process known as preferential attachment.
